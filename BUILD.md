Build
===

To build `its-a-twitter` is pretty simple.

# API

Was developed with Go 1.18. Might work for older versions.

## Run (Development)
```bash
cd api/
go run . # Automatically installs modules
```

## Build
```bash
cd api/
go build .
```

## Testing
```bash
cd api/
go test ./controller/
```


# Frontend

Run:

```bash
npm i
```

## Run (Development)
```bash
cd frontend/
npm start
```

## Build
```bash
cd api/
npm run build
```
